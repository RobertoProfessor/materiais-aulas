Parte 2: Dimensionamento Prático

DESAFIO: Pilar Retangular (Abas Iguais)

Pilar 20x60cm, Pk = 800kN, Solo = 0,25 MPa. Calcule L e B.

1. Area = (800 * 1.1) / 250 = 3.52m²
2. D = 0.60 - 0.20 = 0.40m
3. L² - 0.4L - 3.52 = 0 -> L ≈ 2.08m (Adota 2.10m)
4. B = 2.10 - 0.40 = 1.70m

DESAFIO: Sapata Associada

P1=600kN, P2=900kN. Distância entre eixos=2,0m. Solo=150kPa. Área total?

Area = (600 + 900) / 150 = 10,0 m².
Nota: O CG da sapata deve coincidir com a resultante das cargas.
